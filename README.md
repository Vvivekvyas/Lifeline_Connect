# 🩸 Lifeline Connect

Lifeline Connect is a full-stack blood donor finder web application designed to connect blood donors with patients in need. The platform allows users to register as donors by providing their blood group and location details, while recipients can easily search for suitable donors based on city, state, and blood type.

Built using HTML, CSS, JavaScript, Python (Flask), Sendgrid and MongoDB, the project focuses on creating a simple, fast, and user-friendly solution for emergency blood donation assistance.

✅ Donor Registration System

# 💡 Project Purpose

The main purpose of Lifeline Connect is to make the blood donation process easier and faster during emergencies by digitally connecting donors and recipients through a simple web platform.
